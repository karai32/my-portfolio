# My Portfolio

This is my personal portfolio website built with React and Vite.

## Features
- Multi-language support
- Contact form with PHP backend
- Responsive design

## Installation
1. Clone the repo:  
```git clone https://github.com/karai32/my-portfolio.git```

2. Install dependencies:  
```npm install```

3. Run the project:
```npm run dev```
